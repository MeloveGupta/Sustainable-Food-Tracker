# Sustainable Food Tracker

Live Demo: **https://know-your-plate.netlify.app/**

A simple React + Vite app that helps users make healthier and more sustainable food choices by showing nutrition, eco scores, additives, charts, and recommendations.

## Features
- Food overview (name, image, brand)
- Eco Score & sustainability percentage
- Nutrition summary (calories, macros, sugar)
- Additives list
- Nutrition bar chart (Chart.js)
- Sustainability meter
- Recommended alternatives
- Clean responsive UI

## Tech Stack
- React 18 + Vite  
- React Router  
- Chart.js / react-chartjs-2  
- CSS (Grid & Flexbox)
